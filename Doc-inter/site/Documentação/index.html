Para que haja comunicação entre os roteadores R1 e R2 precisará configurar o IP 2001:db8:acad:4::1/127 da interface 0/1 no R2.

Comando: ipv6 rip rl enable

Função: Habilita o protocolo RIPv6 na interface atual, permitindo que o roteador R1 compartilhe informações de roteamento IPv6 com outros dispositivos da rede. Essa configuração é essencial para que os pacotes IPv6 sejam encaminhados corretamente entre os diversos segmentos da rede.

Esse comando terá que ser feito em todas as interfaces 0/0 , 0/1 e 0/2 de todos os roteadores .

VLAN {#vlan}

VLANs (Virtual Local Area Networks) são uma tecnologia essencial para otimizar e segmentar redes locais, proporcionando maior segurança, desempenho e flexibilidade.

O que são VLANs?

Uma VLAN é uma rede lógica criada dentro de uma rede física, permitindo que você divida sua rede em várias sub-redes virtuais independentes. Cada VLAN opera como se fosse uma rede separada, mesmo compartilhando a mesma infraestrutura física, como switches e roteadores.

As VLANs funcionam atribuindo tags (etiquetas) aos pacotes de dados, identificando a qual VLAN eles pertencem. Os switches, então, encaminham o tráfego apenas para as portas que pertencem à mesma VLAN, garantindo o isolamento e a segurança entre as diferentes redes virtuais.

Cada VLAN é considerada uma rede lógica separada.
